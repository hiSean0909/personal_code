# agent/ — Agent 开发学习与实践

由浅入深地学习 AI Agent 开发，产出两类内容：

- **学习过程**（`my_learn/`）— 在 AI 带领下步步为营，按模块记录代码和笔记
- **自主实践**（`my_practice/`）— 学完每个知识点后独立练手

## 内容

| 文件/目录        | 说明                                                              |
| ---------------- | ----------------------------------------------------------------- |
| `my_learn/`    | 学习过程记录（详见[my_learn/readme.md](my_learn/readme.md)）       |
| `my_practice/` | 自主实践项目（详见[my_practice/readme.md](my_practice/readme.md)） |

---

> **学习方式**：AI 讲概念 → 读代码印证 → 动手写文件 → 跑起来验证 → 独立练习
>
> 每个知识点对应一个独立文件，不在已有文件上修改。
